# Proposal: numeric class stat minimums (2026-07-01)

**Status: awaiting Daniel's review.** Review file:
`design/MMOStats-proposed-stat-minimums.xlsx` — proposed cells are yellow with
blue bold numbers; everything else is untouched. Generated by
`tools/propose_stat_minimums.py`.

## The rule (reverse-engineered from Great North)

Great North's 18 numbered classes follow one rule almost perfectly:

> **class minimum = archetype floor + 2**

Evidence: of 91 numbers on the GN sheet, 89 are exactly floor+2. The only
exceptions are **Volva** (calmness 10, wisdom 10 — signature-stat spikes well
above the +2 rule). Every numbered class keys exactly 5 stats (Volva keys 6).

## What this proposal fills in

283 cells across 57 classes, applying floor+2 (capped at 10) to every x-mark
in the stats section:

- **Great North**: the 7 unnumbered DPS classes (Draoi, Pankrator,
  Frontiersman, Ranger, Relic Master, Whisper, Veil Tamer)
- **Mystic Lands**: all 25 classes
- **Honorguard**: all 25 classes

## Things to review by hand

1. **DPS Logic floor is 1 on all three sheets.** Every other floor is 3–8, so
   this looks like a placeholder. It matters because the classes keying Logic
   are the caster-flavored DPS (Sorcerer, Storm Caller, Illusion, Hydramancer,
   Tempest, Ruin Singer, Wave Master, Charmer, Veil Tamer, Archer) — under the
   rule they get Logic 3, which reads far too low for casters. If the floor
   should be ~7 (matching Frontline/Support), their minimums become 9.
2. **Signature spikes.** GN gave Volva two 10s as signature stats. If other
   classes deserve the same treatment (each faction's mystic healer? each
   faction's iconic class?), that's a hand-tuning pass on top of this
   proposal — the +2 baseline doesn't capture it.
3. **Key-stat counts vary.** GN's numbered classes all key exactly 5 stats;
   the x-marked classes range roughly 3–7. If "exactly 5" is a design
   invariant, some classes need keys added/removed before numbers are final.
