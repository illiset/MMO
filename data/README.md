# data/ — canonical design data

Generated by `python tools/extract_from_xlsx.py` from the MMOStats.xlsx
snapshot in `design/`. **Do not hand-edit** — change the spreadsheet (or the
extractor) and regenerate. Validate with `python tools/validate.py`.

## Files

- `stats.json` — the 16-stat taxonomy (4 categories × 4 stats) and the
  100-point-per-archetype budget
- `factions/<id>.json` — one file per faction: `great-north`, `mystic-lands`,
  `honorguard`
- `abilities.json` — **hand-authored** (the one file NOT generated from the
  spreadsheet; edit it directly). Validated against faction/class ids by
  `tools/validate.py`

## Faction schema

```jsonc
{
  "id": "great-north",
  "name": "Great North",
  "abbreviation": "GN",
  "archetypes": [
    {
      "id": "frontline",
      "name": "Frontline",
      "statMinimums": { "strength": 7, ... },   // archetype-wide floor
      "classes": [
        {
          "id": "knight",
          "name": "Knight",
          "statMinimums": { "strength": 9, ... }, // class-specific, above floor
          "keyStats": []                          // stats flagged x, no number yet
        }
      ]
    }
  ],
  "cultureGroups": [
    {
      "id": "crusaders",
      "name": "Crusaders",
      "races": [
        { "id": "celtic", "name": "Celtic", "allowedClasses": ["knight", ...] }
      ]
    }
  ]
}
```

All cross-references use slug ids (`lowercase-hyphenated`, apostrophes
dropped: `Wei' Vicari` → `wei-vicari`).

## Known quirks inherited from the spreadsheet

- **Great North has numeric class stat minimums; Mystic Lands and Honorguard
  only have `keyStats` markers.** GN is the reference faction; assigning
  numbers to ML/HG is open design work, not an extraction bug.
- Spreadsheet stat name `Widsom` is normalized to `wisdom` (mechanical ids
  must match across factions). Creative names are kept as authored — note
  `Provacateur` (GN Support class) is the spelling on the sheet.
- Column C of each sheet carries leftover scaffolding labels
  (`Government`, `Crusaders`, `Bands`, `Hillback`, `Dew Hollow` — even on
  non-GN sheets). The extractor ignores that column entirely.
- The row of `100`s under each archetype column is read as the shared
  point-buy budget (`budgetPerArchetype` in `stats.json`).
